# Thrift简介

## 安装
安装环境为ubuntu,运行如下命令安装:
```shell
sudo apt-get install automake bison flex g++ git libboost1.55-all-dev libevent-dev libssl-dev libtool make pkg-config
sudo apt-get install libglib2.0-dev

./configure
make 
sudo make install
```
